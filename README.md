 # 广告创意图片打分工具
> 🚀 **[点击此处在线体验工具](https://tomos123456.github.io/Rachel-s-idea/)**
 一个面向信息流广告投手的图片素材智能评估工具。上传广告素材图片，自动计算五维视觉分数，匹配最适配的投放类目，并给出主图/配图定位建议。
 
 ## 功能
 
 - **五维视觉评分**：色彩饱和度、主体参与度、文字覆盖率、留白比例、构图规范性
 - **类目预设权重**：内置 6 个广告类目（电商、App推广、教育培训、游戏买量、美妆个护、健身运动），一键切换，权重自动调整
 - **类目匹配推荐**：基于余弦相似度算法，自动判断图片最适合投放哪个类目
 - **素材定位建议**：分析图片特征，推荐作为主图或配图使用
 - **多图对比**：支持多张素材同时分析，对比评分
 - **即粘即用**：支持 Ctrl+V 粘贴截图，无需保存到本地
 
 ## 使用方式
 
纯前端应用，无需安装任何环境，可直接点击上方链接在线使用。
 
 ```
 截图/下载素材 → Ctrl+V 粘贴 → 自动分析 → 查看评分与建议
 ```
 
 ## 技术实现
 
 - HTML/CSS/JavaScript 纯前端，零依赖（除 Chart.js 雷达图库）
 - Canvas 像素级分析：色彩饱和度、边缘检测、对比度分析、留白检测、九宫格构图评估
 - 余弦相似度算法进行类目匹配
 - 加权规则引擎生成素材定位建议
